mew-prototype
=============

The MEW (Multitenancy-Elasticity Wrapper) is a middleware that converts a simple service into a multitenant-elastic service for transitioning to PaaS.

INSTALLATION INSTRUCTIONS
=========================
1. The MEW is a code template rather than a full solution. Therefore it is suggested that you fork this repository so that you tailor it to your particular needs, without changing the core project.

2. Install the following prerequisites:
•	Ruby 1.9.2p180
•	MySQL server version 5.2
•	Install the following Ruby gems (gem install):
	o	parseconfig
	o	sqlite3 
	o	mysql2 
	o	soap4r
	o	logger
	o	uri
	o	dalli

3. Clone the repository in any directory.
